# Rekordbox

RbLighting only supports certain Rekordbox versions, due to the method it extracts timing data.

The table below outlines the supported versions of Rekordbox.

|Rekordbox Version  |Supported  |
|-------            |:---:      |
| 7.1.5             | ☑        |
| 6.8.5             | ☑        |
| 6.8.4             | ☑        |
| 6.8.3             | ❌       |
